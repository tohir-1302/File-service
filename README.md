# Nest.js File Service Microservice with MinIO Storage

This microservice is designed to facilitate file uploading and downloading functionalities within a Nest.js application, using MinIO, an open-source, high performance object storage service that is compatible with Amazon S3 APIs.

## Features

- **File Upload:** Allows users to upload files to the MinIO bucket.
- **File Download:** Enables users to download files from the MinIO bucket.
- **Security:** Implements security measures to ensure safe file operations.
- **Scalability:** Designed to be scalable with the growth of the application.

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Node.js (LTS version)
- Docker (for running MinIO locally)
- Nest.js CLI
- An IDE or text editor of your choice
